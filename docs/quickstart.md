## 📋 **Setup Completo (30 minutos)**

### **🎯 Passo 1: Definir seu ICP (Ideal Customer Profile)**

#### **Perguntas Essenciais:**
```
❓ Defina seu cliente ideal:
  ├── Qual setor/CNAE você atende?
  ├── Qual o porte ideal das empresas?
  ├── Em quais regiões você atua?
  ├── Qual faixa de faturamento?
  └── Há alguma característica específica?
```

#### **Exemplo Prático:**
```
🎯 ICP - Software de Gestão:
  ├── CNAE: 4781-4 (Comércio varejista)
  ├── Porte: Micro e Pequeno
  ├── Estados: SP, RJ, MG
  ├── Capital: R$ 50k - R$ 500k
  └── Situação: Ativa
```

### **🔍 Passo 2: Configurar Filtros Avançados**

#### **Interface de Filtros:**
```
📍 Localização:
  ├── Estado/UF: [Dropdown com todos os estados]
  ├── Cidade: [Autocomplete baseado no estado]
  ├── CEP: [Range ou específico]
  └── Região: [Norte, Nordeste, etc.]

🏢 Empresa:
  ├── CNAE: [Busca por código ou descrição]
  ├── Porte: [Micro, Pequeno, Médio, Grande]
  ├── Situação: [Ativa, Suspensa, etc.]
  ├── Capital Social: [Range de valores]
  └── Data Abertura: [Período específico]
```

### **📊 Passo 3: Executar Busca e Analisar**

#### **Executar Busca:**
```javascript

## 🎯 **Classificação de Leads**

### **🤖 Ativar Classificador IA**

#### **Configuração Inicial:**
```
⚙️ Configurar IA:
  ├── Acesse: Menu > Classificador IA
  ├── Defina critérios de qualificação
  ├── Configure pesos por característica
  ├── Ative classificação automática
  └── Defina thresholds (Like/Maybe/Dislike)
